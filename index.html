Components

CPU (Central processing unit)
เป็นหน่วยประมวลผลกลาง ทำหน้าที่ประมวลผลข้อมูล ถ้าให้เปรียบเทียบกับมนุษย์ ก็จะคล้ายๆกับสมองของมนุษย์นั้นเอง มีลักษณะเป็นชิพสี่เหลี่ยม
RAM (Random access memory)
เป็นหน่วยความจำ มีหน้าที่เป็นที่พักข้อมูลให้กับCPU หน้าที่ของมันจะคล้ายๆกับกระดาษทด มีลักษณะเป็นแผงสี่เหลี่ยมยาวๆมีลักษณะต่างกันไปในแต่ละยี่ห้อ
Mainboard
เป็นแผงวงจรขนาดใหญ่ที่คอยเชื่อมต่ออุปกรณ์ทุกอย่างเข้าด้วยกันตามชื่อของมันที่เรียกว่าบอร์ดหลัก
Power supply
เป็นแหล่งจ่ายไฟให้กับทุกอุปกรณ์ของคอมพิวเตอร์ มีลักษณะเป็นกล่องสี่เหลี่ยม
HDD (Hard disk drive)
เป็นอุปกรณ์ที่เอาไว้เก็บข้อมูล มีลักษณะเป็นกล่องสี่เหลี่ยมๆ เปรียบเสมือนเป็นคลังเก็บของของคอมพิวเตอร์
VGA Card (Video Graphic Adapter Card)
หรือที่เรียกกันว่าการ์ดจอ ทำหน้าที่ประมวลผลเฉพาะที่เกี่ยวกับภาพ การ์ดจอจำเป็นอย่างยิ่งเมื่อต้องการทำงานหรือทำอะไรก็ตามเกี่ยวกับการประมวลผลภาพ
